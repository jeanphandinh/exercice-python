# Python Exercises

Ce dépôt regroupe mes exercices et projets personnels en Python.

## Objectif
Renforcer mes bases en programmation et appliquer progressivement les compétences travaillées pendant mon apprentissage de Python.

## Contenu du dépôt

### Basics
Exercices sur les notions fondamentales de Python :
- variables et affichage ;
- conditions ;
- boucles ;
- listes ;
- fonctions ;
- entrée utilisateur ;
- dictionnaires ;
- boucle `while` ;
- chaînes de caractères et tuples ;
- exceptions simples.

### Intermediate
Exercices sur des notions plus avancées :
- modules ;
- gestion des exceptions ;
- lecture et écriture dans des fichiers ;
- manipulation de chaînes ;
- list comprehensions ;
- lambda et closures ;
- programmation orientée objet.

### Projects
Projets personnels permettant d’appliquer les notions vues :
- gestionnaire de notes ;
- analyseur de texte ;
- gestionnaire de bibliothèque personnelle.

## Organisation
- `basics/` : exercices de niveau fondamental ;
- `intermediate/` : exercices de niveau intermédiaire ;
- `projects/` : projets plus complets et structurés.

## Démarche
L’objectif de ce dépôt est de montrer ma progression en Python à travers la pratique, avec des exercices simples puis des projets plus complets.

## Remarque
Le dépôt `python-certification` est consacré à mon parcours de formation et à mes certifications, tandis que ce dépôt est dédié à l’entraînement personnel et aux projets.
